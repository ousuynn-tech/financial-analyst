# Профессия: Финансовый аналитик (ЭР-Телеком)

Сайт представляет собой «цифровой портрет» профессии финансового аналитика, созданный для хакатона/конкурса. Наша цель — доступно и увлекательно рассказать первокурсникам, кто такой финансовый аналитик, чем он занимается и почему эта роль критически важна для компании ЭР-Телеком (ЭРТХ).

## 🧠 Использование ИИ
Мы осознанно использовали ИИ для генерации структуры, стилей и контента. Примеры наших промптов:

1. **Генерация базового приложения:**
   > "Создай Next.js приложение с Tailwind CSS и Ant Design. Напиши каркас сайта для профессии 'Финансовый аналитик в ЭР-Телеком', используя современные стили, карточки и отступы."
2. **Генерация контента:**
   > "Перепиши текст о задачах финансового аналитика так, чтобы он был понятен первокурснику. Избегай сложных профессиональных терминов, объясняй на простых примерах (доходы, расходы, тарифы)."
3. **Создание интерактива:**
   > "Напиши React-компонент интерактивного 3D-кубика с помощью Tailwind и встроенных стилей transform: rotate3d. При нажатии на кнопку кубик должен вращаться 2.4 секунды и выдавать случайный факт о профессии."

##  Ручные правки 
ИИ отлично справляется с написанием кода и текстов, но мы вручную:
1. **Упрощение языка:** Вручную переписали блоки "Задачи" и "Кто это", чтобы убрать типичные для ИИ канцеляризмы и сделать текст более "человечным" и простым.
2. **Добавление блока 'Образование':** Добавили секцию об образовании на страницу Skills, так как ИИ сгенерировал только hard/soft навыки.
3. **Настройка адаптивности:** Вручную поправили классы Tailwind (добавили `flex-wrap`, поправили сетки `lg:grid-cols-3` и `md:grid-cols-2`), чтобы карточки корректно отображались на мобильных экранах.

## Цикл разработки (Change -> Commit -> Auto-deploy)
Мы настроили непрерывную интеграцию с использованием **GitHub** и **Vercel**:
1. Разработка ведется локально на ветке `main`.
2. После внесения изменений мы делаем коммит: `git add .` -> `git commit -m "feat: update content"`.
3. Отправляем изменения в репозиторий: `git push origin main`.
4. Vercel автоматически перехватывает пуш, собирает Next.js приложение (`npm run build`) и разворачивает обновленную версию на публичном домене.
